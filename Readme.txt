Zur Installation und Start der Anwendung lesen Sie unter kindergarten-2.1.0/hilfe/Installation.html die Installationsanweisungen.

Für genauere Beschreibungen der Funktionen des Programmes starten Sie die Anwendung über folgenden Pfad:
kindergarten-2.1.0/bin/kindergarten.bat und öffnen im Menü Info den Hilfedialog. Sie können alternativ auch über kindergarten-2.1.0/hilfe/Benutzerhandbuch.html das Benutzerhandbuch öffnen.

Im HowTo ist der Arbeitsablauf mit dem Programm genauer beschrieben. Sie finden es unter kindergarten-2.1.0/hilfe/HowTo.html.
